# @fyndx/react-native-swipe-carousel

Swipe Carousel for react native with reanimated and gesture handler

## Installation


```sh
npm install @fyndx/react-native-swipe-carousel
```


## Usage


```js
import { multiply } from '@fyndx/react-native-swipe-carousel';

// ...

const result = await multiply(3, 7);
```
